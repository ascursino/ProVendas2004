Cadastrando uma venda
 
A tela abaixo é o início do cadastro de uma venda. Uma venda pode estar ou não relacionada a um cliente. Para relacionar um cliente a sua venda, basta clicar 2 vezes no nome do cliente que aparece na listagem, caso esse já esteja cadastrado no sistema, ou clicar no botão INCLUIR CLIENTE para fazer um novo cadastro de cliente e logo após, o cadastro da venda. Caso não seja necessário cadastrar um cliente para uma determinada venda, basta clicar no botão VENDA RÁPIDA, neste caso a venda só poderá ver à vista e não tem a necessidade de se cadastrar um cliente para fazer a mesma.
   
 
   
 

A tela abaixo é um exemplo de cadastro de vendas. Basta preencher os campos mostrados nesta tela e para finalizar a inclusão desta venda, clique no botão OK.

O botão EXCLUIR PRODUTO, ao ser clicado, retira o produto da listagem de venda. Para isto, basta selecionar na listagem o produto que se quer retirar e clicar no botão referente a exclusão deste.

O botão VER PRODUTO, ao ser clicado, mostrará uma listagem com todos os produtos. Para escolher um produto para está venda basta clicar 2 vezes sobre o nome do produto que está na listagem exibida. Como mostrado logo abaixo.

O botão CREDIARISTA aparecerá quando a venda for a prazo e, ao ser clicado, aparecerá uma listagem de crediarista para que seja escolhido um para o financiamento desta venda. Para escolher um crediarista na listagem basta clicar 2 vezes sobre o nome do crediarista escolhido ou se, o crediarista for o próprio cliente, é só marcar o campo O PRÓPRIO.
